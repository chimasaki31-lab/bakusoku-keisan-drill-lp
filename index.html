なぜ計算特訓が必要なのか

受験漫画『ドラゴン桜』が
教えてくれた特訓法

『ドラゴン桜』では、100問の算数プリントを毎日繰り返すことで 計算の処理速度を上げ、受験で戦える基礎力をつける特訓が紹介されています。

計算が速くなると、頭の中に余裕が生まれる。
その余裕が、難しい問題を落ち着いて解く力になる。

このアプリは、そのプリントをスマホでいつでも
実践できるように作りました。

PROBLEM

こんなこと、ありませんか？

  • ⏱ テストで時間が足りず、見直しができない
  • 🔢 わかっているのに計算ミスで点を落とす
  • 📖 文章問題で詰まるのは、計算が遅いから？
  • 📚 算数プリントをやらせたいけど、毎回印刷が面倒

FEATURES

アプリでできること

➕➖✖️➗

4種類の演算を特訓

足し算・引き算・掛け算・割り算をそれぞれ選んで、または全部まとめて練習できます。

🔢

10問・50問・100問

その日の時間や気分に合わせて問題数を選択。毎日無理なく続けられる設計です。

⏱

タイムを自動計測

解答時間をリアルタイムで計測。タイムが縮まると上達がはっきりわかります。

📊

過去20回分の履歴

タイム・ミスの種類・問題数をすべて記録。昨日の自分を超えることが目標になります。

HOW TO USE

使い方は3ステップ

1

種目と問題数を選ぶ

足し算・引き算・掛け算・割り算から1つ選び、10問・50問・100問のどれかを選択します。

2

どんどん答えていく

画面に表示される問題に次々と答えます。スピードを意識して解くことが上達の近道です。

3

タイムと結果を確認

終了後にタイムとミスを確認します。毎日続けると、自分の成長がはっきり見えてきます。

タイムを記録。
昨日の自分を超えよう！

成長が数字で見えるから、毎日続けられます。過去20回分の結果を自動保存。

さあ、計算特訓を
始めよう！
